# Security Policy

## Supported Versions

This project is currently in early development. Only the latest `main` branch is supported.

## Reporting a Vulnerability

If you discover a security issue, please report it privately to the maintainer before public disclosure.

Include:

- A clear description of the issue
- Affected version/commit
- Reproduction steps
- Potential impact

You will receive an acknowledgement and follow-up on remediation steps.
